# Student-Dashboard

## Stack Used
- Frontend (React) using create-react-app
- Backend (Express)
- Databse (MongoDB)
- hosting (Heroku)
- React + Express combined [x]

### Features
- React
- Express
- React-Router
- Webpack
- ES6 Support
- CSS-Loaders
- Hot Reload Module

#### To get started
```shell
$ git clone https://www.github.com/htadg/React-Express-Boilerplate.git <YOUR_PROJECT_NAME>
$ yarn install
```
#### To start a development server
```shell
$ yarn start
```

#### To start a production server
```shell
$ yarn start:prod
```

#### To build the app
```shell
$ yarn build:prod
```
